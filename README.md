## Project Overview
Handcrafted Haven is a web application designed to connect artisans and crafters with customers who value unique, handmade products. It provides a digital marketplace where creators can showcase their craftsmanship, share their stories, and sell their items. The project emphasizes community building, sustainable consumption, and accessibility. The purpose of this project is to provide a platform for artisans to list and sell handcrafted items and foster a community of creators and conscious consumers.

## Tech Stack
- Front-End: HTML, CSS, JavaScript, Next.js
- Back-End: Node.js with a database  
- Project Management: GitHub Boards  
- Code Management: Git & GitHub
